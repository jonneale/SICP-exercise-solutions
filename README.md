Emacs Shortcuts
---------------

*inferior-lisp*:
        C-c C-e Evals expression with inferior lisp interpreter


SLIME/SWANK:
        C-x C-e Evals expression in SLIME. When used with def provides
        symbols in repl.

REPL:
        M-p     Re-loads previously run command
